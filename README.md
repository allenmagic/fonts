# 概述

收录了一些常用的字体，可以用于日常系统的使用。具体的字体可以根据名字进行查看。

其中包括了PingFang，JetBrainsMono和对应的Nerd Fonts。还包括了Fonts Awesome和Material Design Icons等字体。

# 安装字体路径
```bash
$HOME/.local/share/fonts
```

# 使用方法

```bash
git clone --depth=1 https://github.com/allenmagic/fonts.git
cd fonts
chmod +x install.sh
./install.sh
```
